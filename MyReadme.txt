Сделано:
1. Cервер считывает афоризмы из заранее подготовленного файла
2. Cервер и клиент использовует TCP-сокеры для обмена информацией с клиентом
3. Сервер поддерживет несколько соединений одновременно - многопоточный - видно из лога
4. Сервер и клиент  умеет передавать сообщение в сжатом виде (если опция включена) + зашифрованном (шифрование своей функцией не библиотечной)
5. Сервер  идентифициует клиентов по специальному имени (передаваемому на сервер клиентом) - клиет передает в ответном сообщении это имя - см. лог
6. Сервер ведет журнал соединений с клиентами, сохраняя идентификатор клиента и переданное ему сообщение (и в каком виде) - без записи в окно лога

Не делано:
1. Не консольные приложения
2. Сборка не CMAKE
3. Порты для работы клиента и сервера берутся не из  файла настроек
